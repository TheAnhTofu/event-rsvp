# Templates

Các file trong thư mục này là **mẫu tham chiếu**. Khi scaffold project mới, agent nên tạo nội dung trực tiếp trong repo (không bắt buộc copy nguyên file từ đây).

Placeholders:

- `{{PROJECT_NAME}}`
- `{{PROJECT_PURPOSE}}`
- `{{STACK}}`
- `{{PROJECT_SLUG}}` (cho tên file `.mdc`, kebab-case)

Xem repo mẫu đã áp dụng: `event-rsvp` (root `AGENTS.md` + `agent/**`).
